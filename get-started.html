Get Started

Download the YourHealth app.

Choose your platform. Android and Windows download app files after they are uploaded. iPhone opens the iOS install path.

Download for macOS
Android: upload `YourHealth.apk`. Windows: upload `YourHealth-Windows-Setup.exe`. iPhone: Apple requires App Store/TestFlight, or Safari Share → Add to Home Screen.
App Downloads

Android, iPhone, Windows and macOS

Use direct downloads for Android, Windows and macOS. Use iPhone for the iOS install route. All platforms keep the same YourHealth flow.

Android: YourHealth.apk · Windows: YourHealth-Windows-Setup.exe · iPhone: App Store/TestFlight or Home Screen
Deploy Steps
1

Android download

Upload `YourHealth.apk` to this folder, then the Android button downloads the phone app directly.

2

iPhone install

Use TestFlight or App Store for a native iOS app. Until then, Safari can add YourHealth to the Home Screen.

3

Windows download

Build `YourHealth-Windows-Setup.exe` with GitHub Actions, upload it to this folder, then the Windows button downloads it.
